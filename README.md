# Mynavi Job Scraper

マイナビ新卒求人サイトから企業の待遇情報を一括取得し、CSVファイルに出力するスクレイピングスクリプトです。

## 背景

就職活動において、各企業の初任給・残業・休日数・福利厚生などの待遇情報を効率的に比較したいと考え、開発しました。マイナビに掲載されている約25,000社分の情報を自動取得し、Excelで分析できる形式に整形します。

## 取得できる情報

- 会社名・業種・基本情報
- 初任給（複数コースがある場合は平均値を算出）
- 試用期間・残業の有無
- 諸手当・昇給・賞与
- 年間休日数
- 福利厚生
- 受動喫煙防止の取り組み
- 勤務地・勤務時間

## 使用技術

- Python 3
- requests
- BeautifulSoup4
- csv（標準ライブラリ）

## セットアップ

```bash
pip install requests beautifulsoup4 lxml
```

## 使い方

```bash
python scrape_job_website.py
```

実行後、取得したい企業数を入力してください（最大20,000社）。

```
取得したい企業の数を整数で入力 20000以上は不可,テスト時は15
> 100
```

カレントディレクトリに `company_treatment.csv` が出力されます。

## Excelでの読み込み方

文字化けする場合は以下の手順で読み込んでください。

1. Excelを開く
2. データ → データの取得 → テキスト/CSVから
3. エンコードを **UTF-8** に指定して読み込む

## 注意事項

- サーバーへの負荷を抑えるため、リクエスト間に0.5秒のスリープを設けています
- コース別募集・企業情報未公開の企業は自動的にスキップされます
- マイナビの仕様変更により動作しなくなる場合があります
